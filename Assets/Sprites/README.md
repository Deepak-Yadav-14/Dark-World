# Sprites

This folder contains all 2D artwork and textures for the Dark World game.

## Organization

- **Characters/** - Player and character sprites
- **Enemies/** - Enemy sprites and animations
- **Environment/** - Background and environment art
- **UI/** - User interface graphics
- **Items/** - Collectibles and interactive objects

## Art Guidelines

- Maintain consistent pixel art style
- Use appropriate pixels per unit (typically 16 or 32)
- Keep consistent color palette
- Optimize sprite sizes for performance
